# DopamineCut DRD

본 폴더는 'DopamineCut(도파민컷)' 프로젝트의 전체적인 설계 및 기획을 담은 DRD(Design Requirement Document) 문서들을 포함하고 있습니다.

프로젝트의 상세 기획 내용은 아래 각 링크를 통해 확인하실 수 있습니다.

| 문서 제목 (Link) | 주요 포함 내용 |
| :--- | :--- |
| 📄 **[DRD-1. 프로젝트 제안서 및 Specification](./DRD-1.md)** | • 프로젝트 목표, 배경지식 및 핵심 기능 명세<br>• 과제 개발 동기 및 클라이언트 요구사항(개론)<br>• S/W 상세 기술 사양(Specification) |
| 📄 **[DRD-2. 요약문 및 프로젝트 수행 계획](./DRD-2.md)** | • 프로젝트 전체 개요 및 기대효과 요약 (Executive Summary)<br>• Work Breakdown Structure (WBS) 기반 세부 업무 분할<br>• 개발 일정(Gantt 차트) 및 책임 분담(Linear Responsibility 차트) |
| 📄 **[DRD-3. 경쟁 제품 분석 및 수행 방법](./DRD-3.md)** | • 국내/외 유사 서비스 장단점 비교 및 차별화 전략<br>• 시스템 설계 환경 및 개발 도구(S/W, H/W) 운용 방안 |
| 📄 **[DRD-4. 마켓 분석 및 제품 판매 예측](./DRD-4.md)** | • 디지털 헬스케어 및 도파민 디톡스 앱 시장 규모 분석<br>• 타겟 연령층 및 사용자 특성 정의<br>• 향후 예상 판매/다운로드 수치 및 수익 예측 |
| 📄 **[DRD-5. 특허 분석 및 개발비 예측](./DRD-5.md)** | • 국내외 관련 기술 특허 등록 현황 조사 및 회피 전략<br>• H/W, S/W, 인건비 등 시스템 개발 소요 비용 상세 산정 |
