# 2. 전체 기능 소개

## 1. 프로젝트 시스템 개요

 본 시스템은 사용자의 숏폼 사용 패턴을 분석하여 과도한 사용 및 숏폼 콘텐츠 소비를 제어하고, 콘텐츠의 성격에 따라 도파민 점수를 관리하는 지능형 앱이다.
 시스템은 백그라운드에서 숏폼 앱 실행을 감지하고, 이를 두 가지 흐름으로 나누어 처리한다.
> 1. 앱 사용 시간 관리 기능
> 2. 숏폼 콘텐츠 분석 및 제어 기능

 사용자가 숏폼 앱을 실행하면 시스템은 실시간으로 사용 시간을 측정하고, 숏폼 영상 여부를 판단한다.  
 숏폼 영상으로 판단될 경우 화면 캡처 및 OCR 기반 텍스트 추출을 수행하고, 이를 AI 모델에 전달하여 콘텐츠의 카테고리를 분석한다.  
 이후 분석 결과에 따라 도파민 점수를 차감하고 서버와 동기화한다.  
 이 과정을 통해 단순 시간 제한을 넘어, 콘텐츠의 질까지 고려한 사용자 행동 관리가 가능하도록 설계되었다.

## 4. 블록 단위 다이어그램 기반 핵심 기능 설명

> **1. 앱 실행 감지 모듈**  
> 백그라운드에서 유튜브 앱의 패키지를 감지하여 실행 여부를 판단한다. 시스템의 모든 로직은 이 감지를 시작점으로 동작한다.
>
> **2. 사용 시간 관리 모듈**  
> 사용자의 유튜브 이용 시간을 누적 관리한다.
> > 1. 목표 시간의 70% 도달 시 사전 알림 제공
> > 2. 목표 시간 초과 시 3분 간격 오버레이 경고
> > 3. 경고 3회 누적 시 앱 강제 종료
> 
> **3. 숏폼 탐지 모듈**  
> 0.3초 주기로 화면 UI를 분석하여 숏폼 콘텐츠 여부를 판단한다. 숏폼으로 판별될 경우 후속 분석 단계로 진입한다.
> 
> **4. 광고 및 중복 필터 모듈**  
> > 1. 광고 키워드를 기반으로 광고 영상 여부를 판별
> > 2. 영상 ID를 기반으로 기존 시청 여부 확인
> > 이를 통해 분석 대상의 정확도를 높인다.
>
> **5. 시청 의도 검증 모듈**  
> 사용자가 3~5초 사이의 랜덤 시간 동안 영상을 시청했을 경우에만 실제 시청으로 인정한다. 단순 스크롤을 필터링하는 역할을 수행한다.
> 
> **6. 영상 캡처 및 OCR 모듈**  
> > 1. 0.5초 주기로 화면을 캡처
> > 2. OCR을 통해 텍스트 추출
> > 3. 사용자가 다음 영상으로 이동 시 데이터를 전달
> 
> **7. 데이터 정제 및 AI 연동 모듈**  
> > 1. UI 텍스트 및 불필요 문자 제거
> > 2. 중복 단어 병합
> > 3. 정제된 텍스트를 프롬프트와 결합하여 AI 모델에 전달
> 
> **8. AI 분석 및 점수 반영 모듈**  
> > 1. 영상 카테고리 결과 수신
> > 2. 카테고리별 우선순위 기반 도파민 점수 차감
> > 3. 시청 기록 및 점수를 서버와 동기화

## 5. 커뮤니티 기능 (웹 기반으로 진행할 예정)

 본 시스템은 사용자들이 숏폼 사용을 줄이는 과정에서 서로의 경험을 공유하고, 동기부여를 받을 수 있도록 웹 기반 커뮤니티 기능을 제공한다.  
 앱 내부에서는 커뮤니티 기능을 직접 구현하지 않고, 앱 화면 하단 또는 메뉴에 위치한 **커뮤니티 버튼**을 클릭하면 웹 커뮤니티 페이지로 이동하는 방식으로 구성한다.

 커뮤니티 웹 페이지에서는 사용자가 게시글을 작성하고, 다른 사용자의 글을 확인하며, 댓글과 공감 기능을 통해 서로 소통할 수 있다.  
 또한 단순한 자유 게시판뿐만 아니라, 사용자의 관심사와 지역 정보를 기반으로 한 광고성 글이나 추천 글도 함께 제공할 수 있도록 설계한다.

> **1. 앱에서 웹 커뮤니티로 이동 기능**  
> 앱 실행 후 화면 하단의 커뮤니티 버튼을 클릭하면 웹 커뮤니티 창으로 연결된다.  
> 사용자는 앱에서 별도의 복잡한 설정 없이 버튼 클릭만으로 커뮤니티 기능을 이용할 수 있다.
>
> **2. 게시글 작성 기능**  
> 사용자는 웹 커뮤니티에서 자유롭게 게시글을 작성할 수 있다.  
> 게시글에는 숏폼 사용 제한 후기, 목표 달성 경험, 앱 사용 팁, 자기관리 방법, 일상 공유 등의 내용을 작성할 수 있다.
>
> **3. 사진 및 텍스트 업로드 기능**  
> 게시글 작성 시 텍스트뿐만 아니라 사진을 함께 업로드할 수 있다.  
> 이를 통해 사용자는 자신의 목표 달성 화면, 사용 시간 기록, 일상 사진 등을 첨부하여 더 자세한 내용을 공유할 수 있다.
>
> **4. 게시글 조회 기능**  
> 커뮤니티 메인 화면에서는 여러 사용자가 작성한 게시글 목록을 확인할 수 있다.  
> 게시글 목록에는 제목, 작성자, 작성 시간, 댓글 수, 공감 수 등의 정보가 표시된다.
>
> **5. 댓글 작성 기능**  
> 사용자는 다른 사용자의 게시글에 댓글을 작성할 수 있다.  
> 댓글 기능을 통해 서로 응원하거나, 숏폼 사용을 줄이는 방법에 대해 의견을 나눌 수 있다.
>
> **6. 공감 기능**  
> 사용자는 마음에 드는 게시글에 공감을 누를 수 있다.  
> 공감 수는 게시글 목록이나 상세 화면에 표시되며, 사용자들이 어떤 글에 관심을 가지는지 확인할 수 있다.
>
> **7. 내 게시글 확인 기능**  
> 사용자는 자신이 작성한 게시글을 따로 확인할 수 있다.  
> 이를 통해 본인이 작성한 글을 다시 확인하거나, 필요할 경우 수정 및 삭제할 수 있다.
>
> **8. 웹 커뮤니티 내 광고 및 추천 글 기능**  
> 웹 커뮤니티에서는 사용자 연령대나 관심사에 맞는 광고성 글 또는 추천 글을 제공할 수 있다.  
> 예를 들어, 숏폼 제한 앱 사용자층의 관심도가 높은 공부, 운동, 자기관리, 취미 관련 광고나 추천 콘텐츠를 노출할 수 있다.
>
 커뮤니티 기능을 웹 기반으로 구성하면 앱 자체의 구조를 단순화할 수 있고, 게시글, 댓글, 이미지, 공감 수와 같은 데이터를 서버에서 관리하기 쉬워진다.  
 또한 이후에는 인기 게시글, 신고 기능, 사용자 랭킹, 광고 수익화 기능 등을 추가하여 커뮤니티 기능을 확장할 수 있다.
