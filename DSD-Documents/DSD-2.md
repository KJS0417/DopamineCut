# 2. 전체 기능 소개

## 1. 프로젝트 시스템 개요

 본 시스템은 사용자의 숏폼 사용 패턴을 분석하여 과도한 사용 및 숏폼 콘텐츠 소비를 제어하고, 콘텐츠의 성격에 따라 도파민 점수를 관리하는 지능형 앱이다.
 시스템은 백그라운드에서 숏폼 앱 실행을 감지하고, 이를 두 가지 흐름으로 나누어 처리한다.
> 1. 앱 사용 시간 관리 기능
> 2. 숏폼 콘텐츠 분석 및 제어 기능

 사용자가 숏폼 앱을 실행하면 시스템은 실시간으로 사용 시간을 측정하고, 숏폼 영상 여부를 판단한다.  
 숏폼 영상으로 판단될 경우 화면 캡처 및 OCR 기반 텍스트 추출을 수행하고, 이를 AI 모델에 전달하여 콘텐츠의 카테고리를 분석한다.  
 이후 분석 결과에 따라 도파민 점수를 차감하고 서버와 동기화한다.  
 이 과정을 통해 단순 시간 제한을 넘어, 콘텐츠의 질까지 고려한 사용자 행동 관리가 가능하도록 설계되었다.

## 4. 블록 단위 다이어그램 기반 핵심 기능 설명

> **1. 앱 실행 감지 모듈**  
> 백그라운드에서 유튜브 앱의 패키지를 감지하여 실행 여부를 판단한다. 시스템의 모든 로직은 이 감지를 시작점으로 동작한다.
>
> **2. 사용 시간 관리 모듈**  
> 사용자의 유튜브 이용 시간을 누적 관리한다.
> > 1. 목표 시간의 70% 도달 시 사전 알림 제공
> > 2. 목표 시간 초과 시 3분 간격 오버레이 경고
> > 3. 경고 3회 누적 시 앱 강제 종료
> 
> **3. 숏폼 탐지 모듈**  
> 0.3초 주기로 화면 UI를 분석하여 숏폼 콘텐츠 여부를 판단한다. 숏폼으로 판별될 경우 후속 분석 단계로 진입한다.
> 
> **4. 광고 및 중복 필터 모듈**  
> > 1. 광고 키워드를 기반으로 광고 영상 여부를 판별
> > 2. 영상 ID를 기반으로 기존 시청 여부 확인
> > 이를 통해 분석 대상의 정확도를 높인다.
>
> **5. 시청 의도 검증 모듈**  
> 사용자가 3~5초 사이의 랜덤 시간 동안 영상을 시청했을 경우에만 실제 시청으로 인정한다. 단순 스크롤을 필터링하는 역할을 수행한다.
> 
> **6. 영상 캡처 및 OCR 모듈**  
> > 1. 0.5초 주기로 화면을 캡처
> > 2. OCR을 통해 텍스트 추출
> > 3. 사용자가 다음 영상으로 이동 시 데이터를 전달
> 
> **7. 데이터 정제 및 AI 연동 모듈**  
> > 1. UI 텍스트 및 불필요 문자 제거
> > 2. 중복 단어 병합
> > 3. 정제된 텍스트를 프롬프트와 결합하여 AI 모델에 전달
> 
> **8. AI 분석 및 점수 반영 모듈**  
> > 1. 영상 카테고리 결과 수신
> > 2. 카테고리별 우선순위 기반 도파민 점수 차감
> > 3. 시청 기록 및 점수를 서버와 동기화
