# 1. 개발 요약

## 1.1 목적
본 프로젝트('DopamineCut')는 현대인들의 심각한 문제로 대두된 숏폼(Short-form) 콘텐츠 중독을 예방하고, 지속 가능한 디지털 디톡스를 유도하는 것을 목적으로 한다. 기존의 단순 시간 제한 앱들과 차별을 주고자 다음의 목적을 달성하고자 한다.
* **사용자 맞춤형 통제:** 회원가입 시 실시하는 간단한 설문 조사를 바탕으로 유저별 '해로운 카테고리'와 '유익한 카테고리'를 분류하여 순위별로 차등 벌점제를 적용한다.
* **게이미피케이션(Gamification)을 통한 동기부여:** '열품타' 앱을 벤치마킹한 모임(방) 시스템을 통해 유저 간 실시간 랭킹 경쟁 및 상호작용(찌르기 알림)을 유도하여 지속적인 참여를 이끌어낸다.
* **생산성 유지와 커뮤니티의 분리:** 앱 내에서는 숏폼 차단 및 몰입이라는 본연의 기능에 집중하게 하고, 타 유저들과의 소통은 독립된 웹(Web) 커뮤니티 플랫폼으로 분리하여 서비스의 모순을 해결하고 광고 수익을 극대화한다.

이를 통해 현대인의 심각한 숏폼 시청 중독을 예방 및 관리하고 지속적인 시간관리를 통해 유저들에게 더 나은 삶을 살아가게 하는 것이 목적이다.

## 1.2 관련 기술
앞서 DRD-5에서 언급한 특허들을 다시 소개하겠다.
모바일 앱 사용시간 할당 및 차단 시스템, 미션 수행 기반의 애플리케이션 사용 제한 시스템 등이 있고 이 밖에도 앱을 개발하면서 참고한 시중의 '열품타' 앱의 모임(방) 기능과 '당근 마켓'의 광고 기능도 관련 기술 중 하나이다. 아래에 기타 관련 기술을 언급하였으니 참고 하길 바란다. 
* **모바일 화면 제어 및 트래킹:** Android AccessibilityService API를 활성화하여 타깃 앱(YouTube, Instagram, TikTok 등)의 실행 상태 및 포그라운드 UI 이벤트를 실시간 탐지하고 차단 로직을 수행한다.
* **컴퓨터 비전 (OCR):** Google ML Kit Text Recognition 엔진을 활용하여 캡처된 화면 이미지 내 텍스트를 고속으로 추출 및 정제한다.
* **AI 기반 콘텐츠 분류:** 정제된 텍스트와 프롬프트를 결합하여 Gemini API를 통해 시청 중인 숏폼 영상의 카테고리(9종)를 실시간으로 분류 및 판별한다.
* **실시간 데이터 동기화:** Firebase Realtime Database 및 Cloud Messaging(FCM)을 활용하여 모임 내 유저 간 랭킹 변동 및 푸시 알림(찌르기 기능)을 실시간으로 동기화한다.

## 1.3 개발 환경
* **Client (Android App):** Android Studio / Kotlin (Min SDK 26, Target SDK 34)
* **Web (Community):** React.js, HTML5, CSS3, JavaScript, VS Code
* **Server & DB:** Node.js (Express), Firebase (Authentication, Realtime DB, Cloud Messaging, Storage)
* **AI Engine:** Google Gemini API / Google ML Kit Core
* **형상 관리 및 협업 도구:** GitHub, VS Code

## 1.4 제품 요약
* **제품명:** DopamineCut (도파민컷)
* **제품 형태:** 안드로이드 애플리케이션 및 연동 웹 커뮤니티 플랫폼
* **핵심 기능:** 1. OCR 및 Gemini API 기반 숏폼 카테고리 자동 자동 분류 및 차등 벌점 시스템
  2. 그룹 멤버 간 실시간 순위 경쟁 및 '찌르기' 푸시 알림 기능 (앱 내 구현)
  3. 당일 도파민 성적에 따라 유저 글자 크기가 동적으로 변하는 웹 기반 커뮤니티 공간
  4. 웹 내 배너 광고 및 스토어 아이템(확성기, 찌르기권) 판매를 통한 비즈니스 모델(BM) 구축

## 1.5 개발 비용
본 프로젝트는 졸업 과제(프로토타입 개발) 목적으로 진행되므로, 팀원 인건비 등의 직접 기술료는 제외하고 실제 개발 및 서비스 빌드에 투입되는 인프라 비용과 최소 운영비만을 산정하였다. 대부분의 기능을 오픈소스 및 무료 제공량(Free Tier) 범위 내에서 구현하여 비용을 최소화한다.

* **인프라 및 오픈소스 API 비용:**
  - Firebase DB & Hosting: 0원 (Google 프리티어 범위 내 사용)
  - Google ML Kit Core (OCR): 0원 (기기 자체 온디바이스 라이브러리로 무료)
  - Google Gemini API 비용: 약 15,000원 (텍스트 전용 연산 및 테스트용 소액 충전)
* **협업 및 개발 도구:** 0원 (GitHub, VS Code Free 플랜 활용)
* **팀 활동비 및 소모품비:** 100,000원 (회의비, 인쇄비, 자료 조사비 등)

## 1.7 목표 시장
* **주 타깃층 (Core Target):** 숏폼(YouTube Shorts, Instagram Reels, TikTok) 시청 중독으로 인해 학업 성적 저하나 집중력 결핍을 겪고 있는 10대~20대 학생 및 수험생.
* **확장 타깃층 (Sub Target):** 자녀의 스마트폰 및 도파민 중독을 올바른 방향으로 제어하고자 하는 학부모 층, 업무 시간 내 몰입 및 딥 워크(Deep Work) 생산성 향상을 원하는 30대 직장인.
* **시장 진입 전략:** 소셜 경쟁(대장 칭호, 놀림 메시지 등) 요소와 지역 기반/관심사 기반 웹 커뮤니티 광고 시스템(당근마켓 모델)을 결합하여, 유저 스스로 앱을 홍보하고 유입되는 선순환 구조를 구축함. 또한 기존의 단순 시간제한 앱에 그치지 않고 각종 추가 기능들(모임, 커뮤니티, 아이템 제도)을 도입해 유저들 간 서로 경쟁하며 지속적으로 숏폼 시청 시간을 관리하고 앱을 사용하게 할 것이다.
